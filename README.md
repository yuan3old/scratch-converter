# scratch-converter
scratch-converter 是一个对用户友好的 Scratch 作品转换器，可以将您的 Scratch 作品转换为 HTML 网页格式或 Windows 系统中的可执行文件。

## 运行

```
yarn
yarn serve
```

## 构建
```
yarn build
```
